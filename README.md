Простой калькулятор, поддерживающий основные математические операции, специальные значения и использование предыдущего результата.

## Возможности

- Базовые операции: сложение (+), вычитание (-), умножение (*), деление (/), возведение в степень (^)
- Поддержка скобок для изменения порядка вычислений
- Специальные значения: `inf`, `-inf`, `nan`
- Использование предыдущего результата через символ `_`
- Обработка математических неопределенностей (0/0, ∞/∞ и т.д.)
- Константы: `pi` (π), `e` (число Эйлера)

### Требования
- Python 3.6 или выше
- pip (менеджер пакетов Python)

### Использование виртуального окружения (venv) с requirements.txt

1. Клонируйте репозиторий:
```bash
git clone <URL репозитория>
cd calculator
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
```

3. Активируйте виртуальное окружение:
   - **Windows**:
   ```bash
   venv\Scripts\activate
   ```
   - **macOS/Linux**:
   ```bash
   source venv/bin/activate
   ```

4. Установите зависимости:
```bash
pip install -r requirements.txt
```

### Использование uv (альтернатива pip)

Если у вас установлен [uv](https://github.com/astral-sh/uv):

```bash
# Клонирование репозитория
git clone <URL репозитория>
cd calculator

# Создание виртуального окружения и установка зависимостей
uv venv
uv sync

# Активация виртуального окружения
# Windows:
.venv\Scripts\activate
# macOS/Linux:
source .venv/bin/activate
```

### Зависимости

Проект использует только стандартные библиотеки Python (`math`, `re`), поэтому файл `requirements.txt` минимален и содержит только информацию о версии Python.

### Запуск калькулятора

```bash
python calculator.py
```

### Завершение работы

Для выхода из программы введите одну из команд:
- `exit`
- `quit`
- `выход`

Или нажмите `Ctrl+C`.
